🍽️ College Canteen Website

Tired of waiting in long canteen lines? We got you!
Built as our 1st year Software Dev project, this website lets college students 
browse the menu, order food, and pay online — all in one place.

✨ Features
- 🧾 Browse the full canteen menu
- 🛒 Place orders online
- 💳 Pay online without cash
- 📦 Track your order status

🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Database: MySQL, MongoDB
